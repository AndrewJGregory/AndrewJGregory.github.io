Tetrist

Tetrist is a classic remake of the arcade game Tetris. My favorite part of this project is that I implemented it with only vanilla JavaScript and DOMination. No external libraries were used. This allowed me to have fine control over every feature without abstracting any logic away. I particularly enjoyed decoupling the piece's drawing onto the board and the downward movement of a piece.

Procify

Built with Rails as a backend and React/Redux as the frontend, Procify is a single page web application that is inspired by Spotify. Conditionally fetching data with React components along with displaying user-friendly errors when erroneous input is entered were exciting parts of this project. Thoroughly planning this project with wireframes, database schema, and a React component hierarchy greatly aided the development process. A minute of planning is worth an hour of debugging.

DOMination

DOMination is a lightweight DOM manipulation library. It uses native DOM API methods to abstract away the nitty-gritty of interacting with raw HTML elements. This provides an intuitive interface to change the DOM. Above, a group of HTML elements can be checked if any of them have a class or if all of them have the same class. From this project I learned many useful native DOM methods that can be faster than other large DOM manipulation libraries.
