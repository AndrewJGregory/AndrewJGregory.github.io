Projects

MessageMe

This is a full stack web app built with Rails, React, Redux, and PostgreSQL inspired by Facebook Messenger. I think that messaging is a critical aspect of technology, so I wanted to build something that incorporated web sockets and real time communication. Despite a relatively simple two page layout, there is a ton of functionality. I love keyboard shortcuts to speed up my daily workflow, so I implemented tabbing through the search bar and list of users, then pressing enter to start chatting with that user and having the message input automatically focus. In the README for this project, I dive into the features and my implementations of them.

Scraping Results

While working at App Academy, I noticed that there was a task that was dreaded by the coaches there because of how mind-numbing it was: sending out the test reports for tests that our students took at the end of the course. After looking over the process that was in place for emailing the reports to all of the students, I also abhorred the same process because of the astronomical amount of time that it took. I decided to automate the entire process. Downloading the test results, compressing them into one file, emailing them out, even deleting all of the reports became a hands-off process that didn't drain our precious time.

Maintain State

While conducting behavioral interviews for App Academy and taking notes in a web page, I accidently refreshed the page. The notes that I took didn't persist and the entire page was cleared. This frustrated me because I had to go back after the interview and fill in all of the notes again. To remedy this, I wrote this chrome extension to persist the notes that I take if the page refreshes or the tab closes. This was my first taste of writing a chrome extension to improve the quality of my work and made me excited about writing more extensions to automate anything I could about my job.

Tetrist

Tetrist is a classic remake of the arcade game Tetris. My favorite part of this project is that I implemented it with only vanilla JavaScript and DOMination. No external libraries were used. This allowed me to have fine control over every feature without abstracting any logic away. I particularly enjoyed decoupling the piece's drawing onto the board and the downward movement of a piece.

DOMination

DOMination is a lightweight DOM manipulation library. It uses native DOM API methods to abstract away the nitty-gritty of interacting with raw HTML elements. This provides an intuitive interface to change the DOM. To the left, a group of HTML elements can be checked if any of them have a class or if all of them have the same class. From this project I learned many useful native DOM methods that can be faster than other large DOM manipulation libraries.
